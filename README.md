# Projet ML: Santé mentale  
**Mental Health Diagnosis & Treatment Monitoring**

##  Description  
Ce projet utilise le dataset **Mental Health Diagnosis and Treatment Monitoring** (Kaggle) pour analyser des données cliniques et comportementales et construire un modèle prédictif permettant d’anticiper l’issue d’un traitement psychothérapeutique.  

Il comprend :  
- Une exploration des données (EDA) — distributions, corrélations, visualisations.  
- Le prétraitement des données (nettoyage, encodage, normalisation).  
- La modélisation (algorithmes ML) pour prédire l’Outcome.  
- L’évaluation des performances des modèles.  
- Une documentation complète (rapport + code + analyses).
